# ⚡ Yisin 快速测试参考

## 🚀 启动应用

在 Xcode 中：
1. 等待项目加载完成（右上角进度条消失）
2. 确保选择了 `Yisin` scheme（左上角）
3. 按 `⌘R` 运行应用

---

```

3. ✅ 成功
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘⌥T` | 打开翻译输入窗口 |
| `⌘Enter` | 开始翻译 |
| `ESC` | 取消/关闭窗口 |
| `⌘,` | 打开设置 |
| `⌘Q` | 退出应用 |

---

## 🐛 如果出现问题

### 快捷键无反应
- [ ] 检查菜单栏是否有 Yisin 图标
- [ ] 查看 Xcode 控制台是否有错误
- [ ] 尝试重新运行应用（⌘R）

### 输入窗口不显示
- [ ] 窗口可能在其他桌面
- [ ] 尝试 `⌘Tab` 切换应用
- [ ] 查看 Xcode 控制台日志

### 翻译失败
- [ ] 检查 API Key 是否正确
- [ ] 检查网络连接
- [ ] 查看错误信息
