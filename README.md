## Philosophy

Yisin is designed to be a **background presence** - deeply integrated into macOS, appearing naturally when needed, respecting your workflow without interruption.

## Features

- **Global Hotkey Translation**: Select text anywhere, press hotkey, get instant translation
- **Floating Light Window**: Elegant, borderless, vibrancy-enabled result display
- **Menu Bar Integration**: Minimal presence with breathing animations
- **Your Own AI**: Use Gemini API for translation
- **Secure Storage**: API keys encrypted in macOS Keychain
- **Custom Prompts**: Fine-tune translation behavior with prompt templates

## Design Principles

- MUJI-inspired minimalism
- Swiss International Typography clarity
- Apple HIG compliance
- Native macOS experience

## Requirements

- macOS 13.0 (Ventura) or later
- Swift 5.9+
- Xcode 15.0+
- Gemini API Key
